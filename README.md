# Lakshmi Srujana Sanaka – Portfolio

A single-page portfolio site built with HTML, CSS, and JavaScript. No build step required.

## How to run

1. **Open in browser (easiest)**  
   Double-click `index.html` or drag it into your browser.

2. **With a local server (optional)**  
   From this folder in a terminal:
   - **Node:** `npx serve .` then open http://localhost:3000  
   - **Python 3:** `python -m http.server 8000` then open http://localhost:8000  

## Contents

- **index.html** – Structure and content (hero, about, education, skills, projects, certifications, achievements, contact)
- **styles.css** – Layout and styling (dark theme, responsive)
- **script.js** – Nav toggle, scroll effects, footer year

## Deploy

Upload the folder (or just `index.html`, `styles.css`, and `script.js`) to any static host (e.g. GitHub Pages, Netlify, Vercel).
